Mantle: Efficient Hierarchical Metadata Management for Cloud Object Storage Services

The Mantle paper presented at SOSP 25 introduces a novel hierarchical Namespace metadata management approach for cloud object storage services. The open-source GitHub repository containing the prototype implementation code is scheduled for release in 2026.
